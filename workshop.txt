Original mod is not up to date. Code rework probably makes this obsolete anyway.

3.4 compatible. Might work on other versions too.

You were not able to select the special trait for cryogenic species with the frozen Planetary Diversity worlds, so this patch fixes that.

Overwrites one species traits file, which includes the other traits, but there's no other way to do it with stellaris code. 

Requires [url=https://steamcommunity.com/sharedfiles/filedetails/?id=2696518608]Cryogenesis[/url] and [url=https://steamcommunity.com/sharedfiles/filedetails/?id=819148835]Planetary Diversity[/url].
